# Joy2Win-vgamepad GUI 使用说明

## 🚀 快速开始

### 启动方式
1. **双击运行** `start-gui.bat`
2. 或命令行运行：`python gui.py`

## 📋 界面功能

### 1️⃣ 手柄状态区域
- **左手柄**：显示连接状态和电量
  - ⚪ 灰色 = 未连接
  - 🟢 绿色 = 已连接
  - 电量条：0-100%
  
- **右手柄**：同上

### 2️⃣ 配置区域
- **手柄模式**：
  - 双手柄模式（默认）
  - 仅左手柄
  - 仅右手柄

- **方向**：
  - 垂直（默认）
  - 水平（仅单手柄模式有效）

- **高级选项**：
  - ✅ 启用 DSU 服务器（体感映射到模拟器）
  - ✅ 启用手柄控制鼠标

### 3️⃣ 控制区域
- **启动/停止按钮**：
  - 🟢 绿色 = 可以启动
  - 🔴 红色 = 正在运行
  
- **启用震动反馈**：勾选后启用游戏震动

### 4️⃣ 日志输出区域
- 显示程序运行日志
- 显示手柄连接信息
- 显示错误和警告
- **自动显示终端输出**

## 🎮 使用步骤

### 步骤 1：配置
1. 选择手柄模式（双手柄/单手柄）
2. 选择方向（垂直/水平）
3. 根据需要勾选 DSU 或鼠标模式
4. 勾选"启用震动反馈"

### 步骤 2：启动
1. 点击"启动 Joy2Win"
2. 按钮变红色表示运行中
3. 日志显示"正在启动 Joy2Win..."

### 步骤 3：连接手柄
1. 确保 Joy-Con 手柄已开机
2. 等待几秒钟
3. 状态从"未连接"变为"已连接"
4. 电量条显示实际电量

### 步骤 4：开始游戏
1. 打开游戏或 Steam
2. 系统识别为 Xbox 360 手柄
3. 可以直接使用

### 步骤 5：停止
1. 点击"停止 Joy2Win"
2. 或关闭窗口（会提示确认）
3. 状态重置为未连接

## ⚙️ 配置说明

### config.ini 文件
GUI 会自动保存配置到 `config.ini`：

```ini
[controller]
controller = 0  ; 0=双手柄，1=左手柄，2=右手柄

[orientation]
orientation = 0  ; 0=垂直，1=水平

[dsu]
enable_dsu = false
dsu_port = 26760

[mouse_mode]
mouse_mode = 0  ; 0=禁用，1=启用
```

## 🔧 常见问题

### Q: 手柄一直显示"未连接"？
**A:**
1. 确保手柄已开机（按任意键）
2. 确保电脑蓝牙已开启
3. 重启程序再试
4. 检查配置是否正确

### Q: 电量显示不准确？
**A:**
- 未连接时显示 0%
- 连接后显示实际电量
- 电量会每秒更新
- 低于 30% 显示橙色
- 低于 10% 显示红色 + 弹窗警告

### Q: 日志不显示信息？
**A:**
- 确保已启动程序（点击"启动 Joy2Win"）
- 日志会自动滚动到底部
- 查看是否有错误信息

### Q: 没有震动反馈？
**A:**
1. 检查"启用震动反馈"是否勾选
2. 确保游戏支持手柄震动
3. 在 Windows 游戏控制器设置中测试

### Q: 如何切换手柄模式？
**A:**
1. 在"配置"区域选择模式
2. 点击"启动 Joy2Win"
3. 配置会自动保存

## 📊 状态指示

### 连接状态
- ⚪ 灰色圆点 = 未连接
- 🟢 绿色圆点 = 已连接

### 电量颜色
- 🟢 绿色条 = 30% 以上
- 🟠 橙色条 = 10-30%
- 🔴 红色条 = 10% 以下

## 💡 提示

1. **系统托盘**：
   - 最小化后显示在托盘
   - 双击托盘图标恢复窗口

2. **配置保存**：
   - 每次启动时自动保存配置
   - 下次启动会记住上次的选择

3. **日志查看**：
   - 可以复制日志内容
   - 自动滚动到底部
   - 显示所有终端输出

4. **电量警告**：
   - 电量低于 10% 时弹窗
   - 请及时充电

## 🎯 打包成 EXE

### 步骤：
1. 运行 `build-exe.bat`
2. 等待打包完成
3. 在 `dist` 目录找到 `Joy2Win-vgamepad.exe`
4. 复制 `config.ini` 到 `dist` 目录
5. 运行 EXE 即可使用
